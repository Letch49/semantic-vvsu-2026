# Задача 1: Лингвистика и синтаксический разбор текстовых данных

## Описание задачи

В данной задаче вам предстоит выполнить лингвистический анализ текстовых данных с использованием библиотек `natasha` и `pymorphy`. 

### Цель задачи

Найти в текстах **прилагательные-сказуемые**, построить зависимость их совместных употреблений с подлежащими и визуализировать результаты.

## Датасет
Использовать датасет собранных новостей, оставшийся с прошлого семестра (либо найти на kaggle)

### Что нужно сделать

1. **Загрузить текстовые данные** из предоставленного файла
2. **Разделить каждое предложение** на подлежащее и сказуемое
3. **Найти прилагательные-сказуемые** (например, "дом красивый", "погода хорошая")
4. **Построить зависимости совместных употреблений**:
   - Какие подлежащие сочетаются с какими прилагательными-сказуемыми
   - Подсчитать частоту таких сочетаний
5. **Визуализировать результаты** (графики, таблицы, диаграммы)

### Технические требования

#### Разрешенные библиотеки:
- `pandas` - для загрузки файлов.
- `natasha` - для синтаксического разбора и извлечения зависимостей
- `pymorphy` - для морфологического анализа (и предообратки)
- `collections.Counter` - для подсчета частот (подсказка!)
- Стандартные библиотеки Python (json, re, и т.д.)
- Библиотеки для визуализации (matplotlib, seaborn, pandas)
- `библиотека визуализаций` - любая

#### Что нужно реализовать:

1. **Функцию для синтаксического разбора**
   - Использование `natasha` для получения синтаксических зависимостей
   - Выделение подлежащего и сказуемого в каждом предложении

2. **Функцию для поиска прилагательных-сказуемых**
   - Использование `pymorphy` для определения части речи
   - Фильтрация прилагательных, которые являются сказуемыми

3. **Функцию для построения зависимостей совместных употреблений**
   - Создание пар (подлежащее, прилагательное-сказуемое)
   - Подсчет частот с помощью `collections.Counter`
   - Сохранение результатов в удобном формате

4. **Функцию для визуализации**
   - Построение графиков частот
   - Визуализация наиболее частых сочетаний
   - Любые другие визуализации, которые помогут понять данные

- **2 балла**: Построение зависимостей совместных употреблений и подсчет частот
